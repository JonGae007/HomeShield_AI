# HomeShield AI
Ein fortschrittliches KI-basiertes Smart-Home-Sicherheitssystem, das die Sicherheit durch Gesichtserkennungstechnologie und Bewegungssensoren revolutioniert.

## Über das Projekt
HomeShield AI zielt darauf ab, die Heimsicherheit durch den Einsatz von künstlicher Intelligenz zu verbessern. Durch die Verwendung von lokal installierten Sensoren und Kameras wird eine kontinuierliche Überwachung gewährleistet. Das System erkennt bekannte und unbekannte Personen und reagiert automatisch auf ungewöhnliche Aktivitäten. Die Nutzer können das System über eine benutzerfreundliche App steuern, die Echtzeit-Benachrichtigungen und Steuerungsfunktionen bietet.

## Hauptmerkmale
- **Echtzeit-Gesichtserkennung**: Identifizierung von registrierten und nicht-registrierten Personen in Echtzeit.
- **Bewegungserkennung**: Erfassung von Bewegungsdaten zur Analyse von Aktivitäten innerhalb des Hauses.
- **Benutzerfreundliche App**: Kontrolle und Überwachung des Systems von überall aus.
- **Datenschutz und Sicherheit**: Strengste Datenschutzmaßnahmen zum Schutz der Nutzerdaten.

## Technologien
- Python
- TensorFlow/Keras für maschinelles Lernen
- Raspberry Pi für Sensorintegration
- OpenCV für Bildverarbeitung
